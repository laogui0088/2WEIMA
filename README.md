# 企业信息二维码生成器

一个简单实用的二维码生成工具,专门用于生成包含企业信息查询链接的二维码,扫码后可自动填充网站查询栏。

## ✨ 功能特点

- 🎯 **自动填充**: 扫码后自动打开网页并填充统一社会信用代码
- 🏢 **支持国家企业信用信息公示系统**: 内置官方网站支持
- 🔧 **自定义网址**: 可配置任意支持URL参数的查询网站
- 📱 **移动友好**: 响应式设计,手机电脑都能用
- 💾 **下载二维码**: 一键下载生成的二维码图片

## 🚀 使用方法

1. **打开页面**: 直接在浏览器中打开 `index.html`
2. **选择网站**: 
   - 国家企业信用信息公示系统(默认)
   - 自定义网址(需包含 `{CODE}` 占位符)
3. **输入代码**: 填入18位统一社会信用代码
4. **生成二维码**: 点击"生成二维码"按钮
5. **使用二维码**: 
   - 扫码直接访问
   - 或下载保存使用

## 📝 示例

### 国家企业信用信息公示系统
- 输入: `913201177568797564`
- 生成URL: `http://www.gsxt.gov.cn/index.html?uniscid=913201177568797564`

### 自定义网址示例
- 模板: `https://example.com/search?q={CODE}`
- 输入代码: `913201177568797564`
- 生成URL: `https://example.com/search?q=913201177568797564`

## 🛠️ 技术说明

- 纯HTML + JavaScript实现,无需服务器
- 使用 [QRCode.js](https://github.com/davidshimjs/qrcodejs) 生成二维码
- 二维码纠错级别: H (高)
- 尺寸: 256x256 像素

## 💡 URL参数说明

二维码包含完整的查询URL,格式为:
```
网站地址 + 查询参数 + 企业代码
```

扫码后浏览器会:
1. 打开目标网站
2. 自动通过URL参数传递企业代码
3. 网站读取参数并自动填充查询栏
